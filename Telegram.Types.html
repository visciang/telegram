    bot_opts()

    @type bot_opts() :: [
  token: token(),
  max_bot_concurrency: max_bot_concurrency(),
  allowed_updates: [String.t()]
]

    Bot options.

    • token: bot token
    • max_bot_concurrency: maximum number of bots that can run concurrently. Defaults to :infinity.
    • allowed_updates: list of allowed updates (ref: https://core.telegram.org/bots/api#getupdates). Defaults to [].
